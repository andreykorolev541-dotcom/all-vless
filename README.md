# VLESS Subscription Smart Aggregator (IPv4 + IPv6)

Умный манифест-агрегатор VLESS-конфигураций. Проект обходит любые блокировки IP-адресов со стороны облачных дата-центров GitHub Actions, отдавая VPN-клиенту прямые динамические указатели на крупнейшие и ежедневно обновляемые базы прокси (`igareck`, `mahdibland`, `yebekhe`).

Ваш VPN-клиент скачивает этот манифест и сам разворачивает его в сотни рабочих конфигураций.

## ⚙️ ВАЖНО: Настройка в Hiddify

Поскольку наш репозиторий выдает список других подписок, в приложении Hiddify нужно включить одну галочку, чтобы оно зашло внутрь каждой ссылки:

1. Скопируйте ссылку выше.
2. В Hiddify нажмите **Новый профиль** (или **+** в верхнем углу).
3. Выберите **Добавить подписку по URL** (Add из URL).
4. Вставьте ссылку.
5. **Критически важно:** Перед нажатием кнопки «Сохранить», найдите в этом же окне переключатель **"Распознавать как подписку"** (Force as Link / Sub-link parsing) и включите его!
6. Нажмите **Сохранить** и обновите профиль.

---

## 🔄 Как это работает

* **GitHub Actions:** Каждые 6 часов проверяет актуальность зеркал и формирует чистый манифест.
* **Hiddify:** Обращается к вашему файлу `vless_plain.txt`, видит там список источников и напрямую загружает прокси с вашего устройства без блокировок.
